# Cashier Setup

Go to **Surge POS → Cashier → New**. Fill employee, display name, 4–6 digit PIN, and POS Profile. Save.

**Start shift:** tap your name on the picker screen → enter PIN → sell screen opens.

**Switch cashier:** tap the lock icon in the navbar, pick the next cashier, enter PIN. Cart is preserved.

**Log out terminal:** on the picker screen tap **Log out** (bottom-right) → confirm. This ends the Frappe session; the next user must sign in with Frappe credentials.
