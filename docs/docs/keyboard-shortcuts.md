# Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| `/` or `Ctrl+F` | Focus search box |
| `Escape` | Clear search / close dialog |
| `F2` | Open payment dialog |
| `F3` | Hold order |
| `F4` | Void order |

**Barcode scanner** — keep the search box focused; scanned input is matched by barcode and the item is added automatically.
