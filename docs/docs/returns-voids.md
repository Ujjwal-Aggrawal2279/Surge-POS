# Returns & Voids

**Void current order** — click **Void** (or **Reset**) in the cart action buttons. All items are removed.

**Return a completed sale** — click **View Orders**, find the invoice, click **Return**, select items/qty, choose refund method.

**Hold** — click **Hold** to park the order. Start a new sale. Retrieve parked orders from **View Orders → Held**.
