# POS Profile

The ERPNext POS Profile links warehouse, price list, taxes, payment methods, and allowed users to a terminal.

One profile per counter is the standard setup. Multiple profiles on the same site are supported (e.g. per shift or per outlet).

**End of day:** create a **POS Closing Entry** in ERPNext (Point of Sale → POS Closing Entry) to reconcile cash.
