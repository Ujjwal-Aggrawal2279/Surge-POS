# Getting Started

Surge POS runs on ERPNext v15. Open `/surge` in any browser on your local network.

**Setup order:**
1. [Install](installation) the app
2. [Configure](configuration) a POS Profile
3. [Add cashiers](cashier-setup) with PINs
4. [Make a sale](making-a-sale)

**Two-identity model:** one Frappe session per terminal, one PIN per cashier. Cashier switches never break the terminal session.
