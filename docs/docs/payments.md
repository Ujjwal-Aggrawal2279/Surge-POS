# Payments

Select a payment method tile (Cash / Card / Points / Deposit / Cheque) before clicking **Grand Total**.

In the payment dialog, confirm the amount and click **Charge**. On success, the invoice is submitted to ERPNext and a confirmation screen shows the invoice number. Tap **New Sale** to clear the cart.

**Offline:** if the server is unreachable the invoice queues locally and auto-submits when connectivity returns.
